What are Databases?

Databases are a collection of information, they are used for creating, manipulating and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or non-relational database. Relationships means a set of our data is related to another section of our data. Meanwhile in a non-relational database there is not a common related set of our data, databases are put into sections of data called schemas, inside each schema each data set is organized into tables, those tables contain columns and rows to house that particular set of data.

Storage

All databases are just a bin of storage. This storage is housed on your hard drive, often times large companies will have floors upon floors of just database storage (think NSA style). Our computer stores information on the hard drive, we are able to create new information, read information, update information and delete information.
